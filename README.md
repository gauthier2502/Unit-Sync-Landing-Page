## üöÄ Two Ways to View This Project

This repository contains **two versions** of the application:

### 1. Static HTML/CSS/JS Version (Recommended for Quick Viewing)
I have converted the React application into a standalone static website. You can view it without installing any dependencies.

- **Location**: `/static_site` folder
- **How to Run**: Simply open `static_site/index.html` in your web browser (Chrome, Safari, Firefox).
- **Features**: 
    - Full fidelity design using Tailwind CSS (via CDN).
    - Interactive Mobile Menu.
    - Testimonial Carousel with touch support.
    - Expandable FAQ section.
    - Simulated AI Assistant Chat.

### 2. React Application (Source Code)
The original source code built with React, TypeScript, and Vite.

- **Prerequisites**: Node.js installed.
- **How to Run**:
    1. Open a terminal in the project root.
    2. Run `npm install` to install dependencies.
    3. Run `npm run dev` to start the local development server.
    4. Open `http://localhost:5173` (or the port shown in terminal).
